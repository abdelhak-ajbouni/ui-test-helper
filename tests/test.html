Try these steps:

  1. Click the UI Test Helper extension icon
  2. Click "Start Inspection" in the popup
  3. Hover over different elements to see them highlighted
  4. Click on any element to see generated testing queries
  5. Copy the queries you want to use
  6. Press ESC to exit inspection mode
